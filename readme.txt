#Установка сайта
На сервере создается база данных. 
Запишите имя БД, имя пользователя БД и пароль пользователя БД.
Корневая папка сайта помещается на сервер. Переименуйте папку именем своего сайта и наберите в адресной строке имя сервера, имя сайта и install, например, localhost/kalinka/install, после чего начнется установка сайта. По окончании установки перейдите в панель управления сайтом.
 Подробнее см. в Руководстве пользователя на сайте 
 Блоки подключаются в административной панели сайта, пункт меню -Общее-главная страница
Моули сайта подразделяются на вариационные - изменяющие свою позицию на странице и стационарные - расположенные в постоянной позиции на странице. 
 Модули сайта подключаются через административную панель сайта - пункт меню -Общее -Модули.
Ядро сайта представляет собой информационный сайт с  подключением к БД и административной панелью.

АДМИН
Для управления общими настройками служит страница общих настроек Общее.
Настройки модулей ведутся через страницу настроек модулей, если таковые имеются.
